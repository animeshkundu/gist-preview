## Quick Start for Agents

1. **Before any work**: Read [00-core-philosophy.md](00-core-philosophy.md)
2. **Before implementing**: Research using [01-research-and-web.md](01-research-and-web.md)
3. **During implementation**: Follow [02-testing-and-validation.md](02-testing-and-validation.md)
4. **For automation**: Reference [03-tooling-and-pipelines.md](03-tooling-and-pipelines.md)

## Key Principles

- **Docs = Code**: No code without documentation
- **90% Coverage**: All code must be tested
- **Research First**: Use web search before implementing
- **Automate Repetition**: If you do it twice, script it
