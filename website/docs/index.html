GistPreview Documentation

Learn how to use GistPreview to transform GitHub Gists into beautifully rendered web pages.

Getting Started

GistPreview makes it easy to preview any public GitHub Gist. Simply paste the URL and click Preview.

Supported URL Formats

  • https://gist.github.com/username/gist_id
  • https://gist.github.com/gist_id
  • gist.github.com/username/gist_id
  • gist_id (raw 20-32 character hex ID)

Content Types

GistPreview automatically detects the content type of your files and renders them appropriately.

HTML

Full webpage rendering with CSS and JavaScript injection from multi-file gists.

Markdown

GitHub-flavored markdown with syntax highlighting and dark theme styling.

React/JSX

Live React applications with automatic Babel transpilation. Just write your component!

JSON

Syntax-highlighted JSON with color-coded keys, strings, numbers, and booleans.

CSS & JavaScript

Code view with line numbers, language badges, and hover highlighting.

Features

Responsive Preview

Toggle between desktop (100%), tablet (768px), and mobile (375px) viewports to test responsive designs.

Shareable Links

Generate permanent URLs with the format: ?gist={id}&file={filename}

Screenshot Capture

Capture screenshots of your rendered gists and copy them to clipboard or download as PNG.

Fullscreen Mode

Immersive fullscreen preview. Shared links open in locked fullscreen mode for distraction-free viewing.

API Limits

GistPreview uses the public GitHub API:

  • Rate limit: 60 requests per hour (unauthenticated)
  • Gist types: Public gists only
